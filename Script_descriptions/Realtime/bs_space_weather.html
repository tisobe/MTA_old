How to Run the Script(s)

  1. go to: cd /data/mta4/space_weather/
  2. run: ./G13_calc.csh
  3. run: ./G13_Ecalc.csh
  4. run: ./G15_calc.csh
  5. run: ./calculate.csh
  6. run: ./calculate_fast.csh
