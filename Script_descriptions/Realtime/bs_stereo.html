STEREO

Description

This set of scripts copies data and plots from NOAA site and display the data and plots on: http://cxc.cfa.harvard.edu/mta/RADIATION/STEREO2/stereo.html/

NOAA Space Weather Predition Center site is: http://legacy-www.swpc.noaa.gov/stereo/. Note that side B is currently not working.
