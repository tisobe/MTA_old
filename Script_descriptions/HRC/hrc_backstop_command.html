HRC Backstop Command List and HRC HK values

Description

This set of scripts extract command sequence from backstop log (planned commands), and actual conditions from HRC HK data, and compare them to see whether there are any discrepancies. If there are, one of the script sends out email warning to notify the discrepancies.
