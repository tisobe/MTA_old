MTA Envelope Trending of MSID Values

This page provides links to the time trend plots of msid values which MTA is monitoring.

Each data point on the plot is a median point of the data over about 10 day period. Since the input data have 5 minutes collection period for the Main Trend and 1 day for the Secondary Trend, the each data point on the plots of the former categories is an average of about 2,900 data points, but those in the latter categories is around 10 data points.

In the following example plot, the blue-- area indicates the envelope around the data. You can think that this is the range where the data points scattered around, if the data were not averaged out for each data point. If average values are used instead of the median values, this area could be more symmetrically distributed around the data points.

The yellow area indicates the yellow violation, and, the red--- area indicates the red violation. The green dash lines indicate top and bottom envelope prediction to the future.

If you hover your mouse over one of the data points, you can display more information about that data collection period. If you like to magnify the plot, click the magnifier icon at the bottom left corner of the plot and select area that you want to expand. If you like to move around the plot, choose the cross icon, then hold down the button on the plot and drag around the plot. You can go back to the full view by clicking the house icon.

Although it is not shown on the example plot below, if you open a page from links below, there is a link to open a table listing history of violation range changes.
