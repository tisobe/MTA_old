Sorry! Page Was Not Created!

The requested interactive page was not created. There are a few possible reasons.

  1. The requested data are too large and the system could not handle.
    • You can try again; it could be the cpu is less taxed when the time you try again.
    • Reduce the time span and/or the sample size.
  2. There were no data in the requested period.
    • Although this is a less likely case, you can try to see whether the different period will create the plot. If it does, please report to tisobe@cfa.harvard.edu.
  3. The data are not in the SOT engineering database.
    • The interactive plot can be produced only when the data are in SOT engineering database. This is the major causes of not producing the interactive plot.
    • A msid with condtions (such as HRC with HRC-I is in use) are not in the database and computed data (such as HRMA computed average) are not in the database.

If you have any questions, please contact tisobe@cfa.harvard.edu.
